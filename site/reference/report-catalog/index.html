Report Catalog¶

Run reports with:

uv run perceptor --root ROOT report REPORT_NAME --case CASE_ID --format table

Most reports support json, table, csv, and md, but some specialized reports have narrower options. Use uv run perceptor report REPORT_NAME --help for the exact switches.

This catalog is generated from the current report --help command set and groups all 228 report subcommands by topic.
